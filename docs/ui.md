# UI Guide

## Design direction

The interface is intentionally:
- minimal
- toy-like
- educational
- icon-first

## Main playback layout

The playback screen shows:
- the **current action** as one large card
- the **next up to 5 actions** in a smaller row
- a progress bar at the bottom

This is easier for children to read than showing all 12 actions at once.

## Action colors

- Forward -> green
- Backward -> orange
- Turn left -> blue
- Turn right -> purple

## Action icons

The icons use:
- rounded arrow shapes
- simple paw markers
- strong color blocks
- centered composition

The goal is to stay understandable even with little or no text.
